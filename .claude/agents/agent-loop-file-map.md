Reference for the agent-loop worker and judge. Describes what each file does, how they connect, and where to look for specific bug types.

## Skills (`.claude/skills/`)

15 domain skills loaded by the agent on demand. Each contains a `SKILL.md` with domain-specific rules, examples, and schemas. Skills are loaded via `read_file` tool call. Combat games typically load: `weapons` + `vfx-presets` + `game-modules` + `triggers`. Decorative scenes load: `attachments` + `vfx-presets`. The shared `vfx-presets` library is referenced by every skill that needs visuals (replaces the old `spawn-vfx` which conflated VFX with spawn-points).

| Infrastructure (session crashes, agent errors)       | `deep_agent_client.py`, `streaming_session.py` | Agent configuration, error handling                                  |

| UX (chat display, generation cards)                  | `apps/client1/` files                          | React components, SSE parsing                                        |
| Domain knowledge (module config, trigger, widget canon) | `apps/claude-agent/.claude/skills/<domain>/SKILL.md`                       | Skill content                                                        |
| Prompt bloat (too many tokens, slow first turn)      | `system_prompt.py`, `skill_routing.py`         | Reduce prompt size, optimize skill loading                           |
| Mock mode issues                                     | `src/mcp/mock_data.py`, `src/tools/*.py`       | Mock intercepts, preset data                                         |
| Scoring accuracy                                     | `src/eval/*.py`                                | Scorer weights, audit checks                                         |
| Empty-URL prefab orphans / async-callback misses     | `src/tools/game_tools.py:_collect_publish_warnings`, `.claude/skills/audio-3d-models/SKILL.md` (per-tool verification blocks), `src/eval/game_def_audit.py:mesh_prefab_with_empty_modelUrl_and_instance_negligent` | Verification rule, audit thresholds, server-side warnings array |
